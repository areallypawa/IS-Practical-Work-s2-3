# 📚 Стек, очередь и польская нотация

## 📌 Описание проекта

Данный проект посвящён изучению и практической реализации базовых структур данных:

* стек (Stack)
* очередь (Queue) (задание на защите)

А также работе с:

* обратной польской нотацией (Postfix / RPN)
* прямой польской нотацией (Prefix)

---

## 🎯 Цели работы

* Получение практических навыков работы со стеком и очередью
* Изучение принципов работы польской нотации
* Реализация алгоритмов вычисления выражений
* Проведение сравнительного анализа структур данных

---

## ⚙️ Реализовано

### 🧱 Структуры данных

* Собственная реализация стека (на динамическом массиве)
* (опционально) реализация очереди

### 🧮 Алгоритмы

* Вычисление выражений в обратной польской нотации (RPN)
* Вычисление выражений в прямой польской нотации (Prefix)

---

## 💡 Пример

### Обратная польская нотация (RPN)

```
3 4 + 2 *
```

Результат:

```
14
```

---

### Прямая польская нотация (Prefix)

```
+ 3 * 4 2
```

Результат:

```
11
```

## ⚠️ Особенности

* Не используются сторонние библиотеки
* Все структуры реализованы вручную
* Поддерживаются базовые операции: `+ - * /`

## 📊 Сравнение структур

| Структура | Принцип работы | Использование       |
| --------- | -------------- | ------------------- |
| Стек      | LIFO           | Польская нотация    |
| Очередь   | FIFO           | Потоковая обработка |
